HTML Structure

This is one column simple and aesthetic layout inspired by space and quantum mechanics. All text is editable so it is possible to use your custom text and / or change purpose of this template from Error 404 subject to i.e. another common HTTP error, like 401, 403, 500 or 503.

This template uses Georgia and free Google web font - Banger. All text in this page you can easily change in this area:

The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

JavaScript

This theme imports two JavaScript files.

  1. jQuery
  2. Spritely
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Spritely is responsible for background and astronaut animation on this site. If you wanna change e.g. speed or direction of this animation, you should look here for original Spritely documentation.

PSD Files

I've included two fully editable PSD files with this theme.

  1. Space.psd - where you can control appearance of the sky, including black vignette on top and at the bottom.
  2. Lost in Space - Error 404 - where you can edit all content.

Layers are named and well organized.
Speech bubbles are vector shapes, so it's easy to adjust their size to fit them with your custom text.

Example: how to add more links into project.

  1. Open file Lost in Space - Error 404.psd in Photoshop.
  2. Go to menu: Image > Canvas Size... and change it from:

    3. To this:

  3. Turn off layer Speech Bubbles Rasterized.
  4. Turn on folder Speech Bubbles and choose layer Try to Visit BG.
  5. Use Direct Selection Tool or hit A on your keyboard and click on speech bubble you are want to edit. In this case it's Try to Visit BG. You should see:

  6. Push and hold Shift key on your keyboard and start to click in every anchor point on the right side of this path:

  7. Now again push and hold Shift key on your keyboard and hit right arrow key 7 times. It will expand that path for 70 pixels:

  8. Now you can align this path with your search form. To do this simply use Move Tool or hit V key on your keyboard. Click on Search Form folder on your layers palette, push and hold Ctrl key on your keyboard and click on Search BG layer inside Speech Bubbles folder. You should see:

  9. With your Search Form and Search BG selected push and hold Shift key and again hit right arrow key 7 times:

  10. Now you can manage with links. Use Horizontal Type Tool or hit T, click on the text you want to edit and add some links:

  11. When you finished, turn off those folders and layers: Search Form, Text and Space. Click with right mouse button on Speech Bubbles folder and choose Duplicate Group.... When it's duplicated, turn off original Speech Bubbles folder and again click with right mouse button - this time on duplicated folder and choose Merge Group. This time use your right mouse button on Speech Bubbles Rasterized layer and choose Copy Layer Style. Click again with right mouse button on your duplicated Speech Bubbles layer and choose Paste Layer Style. You should see:

  12. It's time to save this image. Go to File > Save for Web & Devices.. or hit Alt + Shift + Ctrl + S on your keyboard and choose PNG-24. Click Save and navigate to your website project and replace an astronaut.png file with your new image.

  13. You can close Photoshop now, because it's time for coding part. Go to styles folder and open main.css file in your favourite program. Go to line 41 and change:
